# Imágenes del Proyecto ESCALA

## Logo Principal

**Archivo requerido:** `logo.png`

### Especificaciones:
- **Formato:** PNG con fondo transparente o circular
- **Tamaño recomendado:** 512x512px (se escalará automáticamente)
- **Uso:** 
  - Navbar de todas las páginas internas
  - Favicon (16x16, 32x32, 180x180)
  - Apple Touch Icon
  - Manifest icons

### Diseño del logo:
- Fondo: Círculo azul oscuro (#0f1729)
- Elementos: Dos montañas con franjas horizontales de colores
- Luna: Círculo beige/crema en la parte superior
- Estrellas: Pequeñas estrellas decorativas
- Franjas base: Naranja (#ff8c42) y Rojo (#e84a23)

### Cómo agregar:
1. Guarda la imagen del logo como `logo.png` en este directorio (`public/images/`)
2. El logo se mostrará automáticamente en:
   - Header (navbar) de todas las páginas
   - Pestaña del navegador (favicon)
   - Iconos de iOS/Android cuando se agregue a pantalla de inicio

### Nota:
Si necesitas otros tamaños específicos para favicon, puedes crear:
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` (180x180)
